# Welcome to Sketchbox,<br/>the UI library for [agendadearte.com](https://agendadearte.com)

Sketchbox is an open source component library that implements the 
[agendadearte.com](https://agendadearte.com) website.\
The library is free to use if it can help or inspire your own project:
[GitHub](https://github.com/agendadearte/sketchbox) & [npmJS](https://www.npmjs.com/package/@agendadearte/sketchbox)

## Installation

<Source code={`
npm install @agendadearte/sketchbox
`}/>

## CSS custom properties
Sketchbox includes many CSS custom properties in the styles of every componet 
that allow you to modify the default styles. In the documentation of each 
component you will find the list of supported CSS custom properties.

<Source code={`
.any-css-hash {
  style-prop: var(${buildPropName('component', 'element', 'cssProp')}, style-fallback);
  ...
}
`}/>

## Rubik font
Sketchbox uses the [Rubik](https://fonts.google.com/specimen/Rubik) font family and relies only on the **300**, **400**, and **500** font weights.
You can add fonts to your project with npm via **[Fontsource](https://fontsource.org/)**, or with the **Google Fonts** CDN.

#### Fontsource
<Source code={`
npm install @fontsource/rubik
`}/>

Then you can import it in your entry point like this:

<Source code={`
import '@fontsource/rubik/300.css';
import '@fontsource/rubik/400.css';
import '@fontsource/rubik/500.css';
`}/>

#### Google Web Fonts
To install the Rubik font in your project using the Google Web Fonts CDN, add the following code snippet inside your project's `<head />` tag:

<Source code={`
<link
  rel="stylesheet"
  href="https://fonts.googleapis.com/css?family=Rubik:300,400,500,700&amp;display=swap"
/>
`}/>
